Cara Membuat Web Ini

1

Struktur HTML

Buat kerangka dasar HTML dengan bagian untuk flashcard, kontrol, dan instruksi.

<div class="flashcard" id="flashcard"> <div class="flashcard-front"> <div class="character" id="character">あ</div> </div> <div class="flashcard-back"> <div class="romaji" id="romaji">a</div> <div class="example" id="example">あさ (asa)</div> </div> </div>
2

Styling dengan CSS

Gunakan CSS untuk membuat efek flip 3D pada kartu dan tata letak responsif.

.flashcard-container { perspective: 1000px; } .flashcard { transition: transform 0.8s; transform-style: preserve-3d; } .flashcard.flipped { transform: rotateY(180deg); } .flashcard-front, .flashcard-back { position: absolute; backface-visibility: hidden; } .flashcard-back { transform: rotateY(180deg); }
3

Data Karakter

Siapkan data untuk hiragana dan katakana dalam array JavaScript.

const hiraganaData = [ { character: "あ", romaji: "a", example: "あさ (asa)", meaning: "pagi" }, { character: "い", romaji: "i", example: "いす (isu)", meaning: "kursi" } ];
4

Fungsi JavaScript

Buat fungsi untuk mengubah konten kartu, navigasi, dan flip kartu.

function updateCard() { const currentData = mode === 'hiragana' ? hiraganaData[currentIndex] : katakanaData[currentIndex]; document.getElementById('character').textContent = currentData.character; document.getElementById('romaji').textContent = currentData.romaji; document.getElementById('example').textContent = currentData.example; }
